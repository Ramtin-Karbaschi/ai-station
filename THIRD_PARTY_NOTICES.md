Before distributing, hosting or commercially using a complete AI Station
installation, review the license and acceptable-use terms of:

1. every selected model;
2. every container image;
3. every Python or system dependency;
4. any search provider queried through SearXNG;
5. any document or dataset processed by the system.

The model manifest records technical provenance and checksums, but it is not a
substitute for reviewing each upstream model license.
